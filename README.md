# Lukasmarts Formulae

## How do I install these formulae?

`brew install lukasmarts/formulae/<formula>`

Or `brew tap lukasmarts/formulae` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lukasmarts/formulae"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
